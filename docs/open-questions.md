# Open Questions

Tracks unresolved product and design questions.

## Current Questions

- What is the source of truth for popularity: external APIs or internal scoring?
- How should lyric excerpts be handled legally?
- What exact formula should convert popularity and experimental level into card value?
- What exact genre matchup matrix should be used in battle?
- Is the main mode collection, duel, or both?

## Candidate Additions

- Should daily point caps vary by account level or remain global?
- Should marketplace fees be fixed or dynamic by card rarity?
- Should enigma difficulty scale by selected genre familiarity?
- Should legacy booster events be monthly by default or seasonal only?
- Should returning legacy packs use full duration windows or short 3-7 day windows only?
