# Decisions

Tracks accepted product decisions.

## Accepted Decisions

- Full collection has no hard deck-size limit.
- Festival battles use 10 active slots.
- Same-artist cards can stack into one active slot.
- Same-album cards can stack into one active slot.
- Variants relate to track families, not artists.
- Stack score uses the most powerful card as base.
- Each supplementary stacked card adds +15% to stack value.
- Daily enigmas: 10 per day, each linked to one song.
- Enigma flow: select genre first, then answer the question.
- Enigma can be an AI-generated image clue about a track.
- Solving an enigma grants the related song card.
- Points can be earned from battles, boosters, enigmas, and freemium purchases.

## Operational Baselines

- battle participation: 10 points
- battle win bonus: +15 points
- daily enigma participation: 2 points
- daily enigma solve bonus: +8 points
- daily soft cap example: 300 points/day from gameplay
- marketplace listing fee example: 2%
- marketplace sale tax example: 5%
