# Progression and Enigmas

Defines progression loops and daily enigma mechanics.

## Scope

This document covers:

- daily enigmas
- missions and challenges
- point-related progression links

## Daily Enigmas

- Each day includes 10 enigmas.
- Each enigma is linked to one song.
- Quiz flow: the player first selects a genre to narrow preferences.
- After genre selection, the game serves the question/enigma from that genre pool.
- An enigma can be an AI-generated image inspired by the track.
- The player tries to guess which track the image represents.
- If a player solves an enigma, they earn the related song card.
- Players earn points for participating in enigmas.
- Players earn additional points when they win/solve correctly.

## Other Progression Loops

- Daily missions: find 3 cards of the same genre.
- Themed challenges: collect lyric quotes around one emotion.
- Set challenges: complete an artist line with 1 artist card and 3 songs.
- Event challenges: win a festival battle with at least 3 active combo bonuses.
- Player trading: marketplace or direct trade.
- Point progression: earn points from battles, booster openings, and optional freemium purchases.
